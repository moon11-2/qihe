附v3 改版说明

  • 机身：标准 iPhone 比例（393:852，约 380×824 展示尺寸）、54pt 屏幕圆角、灵动岛、Home 指示条；边框仅 2pt 深色描边，保持展示板轻量感。
  • 法务藏青 #23405F 承担全部"操作与制度"角色：主按钮、发送键、Tab 高亮、法条依据、流程节点、信任说明；朱砂只保留印章与高风险，两色不混用。
  • 制度感三件套落地：① 风险卡标注"涉及条款 · 第 X 条"，与原文条款编号对应；② 每条风险附"法条依据"引用行（藏青底注样式）；③ 审查过程改为四步流程节点：解析文本 → 识别主体 → 条款比对 → 出具报告。
  • 首页入口下新增一行信任说明（逐条审查 · 标注法条依据 · 历史仅本地保存），草案页标题下增加体例注记行。原型中的法条条号为示意占位，接入后端后按实际引用输出。
  • 古风浓度收敛：底色从宣纸黄调收为克制暖白 #F7F5F0，印章、宋体、骑缝线、壹貳叁编号保留；信息架构与功能边界与前版完全一致。
