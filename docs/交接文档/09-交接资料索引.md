## 10. 常用命令

启动 H5：

```bash
npm run dev:h5 -- --host 127.0.0.1
```

类型检查：

```bash
npm run type-check
```

构建 H5：

```bash
npm run build:h5
```

启动 Dify 本地代理：

```bash
npm run dify:proxy
```

同步到 iOS：

```bash
npm run sync:ios
```

同步 Dify 模式到 iOS：

```bash
npm run sync:ios:dify
```

打开 iOS 工程：

```bash
open /Users/lantian/Documents/合同审查/ios/HetongBang/HetongBang.xcodeproj
```
